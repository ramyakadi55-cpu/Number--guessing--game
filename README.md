# 🎯 Number Guessing Game

A simple command-line game built in Python. The computer picks a random number between 1 and 100, and you try to guess it in 7 attempts or less!

This is my first GitHub repo — built while learning Python basics like loops, conditionals, and user input.

## How to Play

1. Clone this repo:
   ```bash
   git clone https://github.com/your-username/number-guessing-game.git
   cd number-guessing-game
   ```

2. Run the game:
   ```bash
   python guessing_game.py
   ```

3. Enter your guesses when prompted. The game will tell you if your guess is too high or too low.

4. You have 7 attempts to guess the correct number. Good luck!

## Example

```
========================================
  Welcome to the Number Guessing Game!
========================================
I'm thinking of a number between 1 and 100.
Can you guess it?

Attempt 1/7 - Enter your guess: 50
Too high! Try again.

Attempt 2/7 - Enter your guess: 25
Too low! Try again.

Attempt 3/7 - Enter your guess: 37
 Correct! You guessed it in 3 attempt(s)!
```
